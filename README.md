# Age calculator :- 
A simple Age Calculator built using HTML, CSS, and JavaScript. It calculates the user's exact age from their date of birth.

 Features
- Calculate age in years
- User-friendly interface
- Input validation

Technologies Used
- HTML
- CSS
- JavaScript

How to Use
1. Enter your birth year.
2. Click the "Calculate" button.
3. View your exact age instantly.

 Future Improvements
- Dark mode
- Age in weeks, hours, and minutes
- Next birthday countdown
